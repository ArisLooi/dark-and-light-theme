# Dark and Light Theme Toggle

A simple React application that allows users to toggle between dark and light themes.

## Table of Contents
- [Introduction](#introduction)
- [Features](#features)
- [Installation](#installation)

## Introduction

This project demonstrates a basic implementation of theme toggling in a React application. Users can switch between dark and light themes using a checkbox.

## Features

- Toggle between dark and light themes
- Simple and clean UI
- Easy to integrate into other projects

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ArisLooi/dark-and-light-theme.git
    ```
2. Navigate to the project directory:
    ```bash
    cd dark-and-light-theme
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the development server:
    ```bash
    npm run dev
    ```
2. Open your browser and navigate to local host.

##Functions
- App: Main component that manages the theme state and renders the form and theme toggle.
- Form: Component that renders the form with buttons.
- Panel: Component that wraps the form elements and applies the theme.
- Button: Component that renders a button with the appropriate theme.
